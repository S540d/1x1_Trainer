# 1x1 Trainer

A multiplication table trainer for children — practice all 12 tables with four game modes.

## Live

[https://s540d.github.io/1x1_Trainer/](https://s540d.github.io/1x1_Trainer/)

## Tech Stack

| Technology          | Role                        |
| ------------------- | --------------------------- |
| React Native + Expo | Cross-platform framework    |
| TypeScript          | Type-safe JavaScript        |
| React Native Web    | Web support                 |
| Hermes              | Optimized JavaScript engine |
| Service Worker      | PWA offline support         |

## Features

- **4 game modes** — result missing, first factor missing, second factor missing, mixed
- **Child profiles** — multiple profiles with individual progress tracking
- **Tablet & foldable support** — adaptive orientation per device class
- **Sound control** — sounds stop immediately when disabled
- **Visual feedback** — green/right, red/wrong answer indication
- **Score system** — track correct answers per session
- **Offline-capable** — works without internet (PWA)
- **Bilingual** — German and English
